18 & 19. Threading & concurrency

import threading
def task(n):
    print(f"Task {n} started")

threads = []
for i in range(5):
    t = threading.Thread(target=task, args=(i,))
    t.start(); threads.append(t)
for t in threads:
    t.join()

# Better way: ThreadPoolExecutor
from concurrent.futures import ThreadPoolExecutor
def square(n): return n*n
with ThreadPoolExecutor(max_workers=5) as ex:
    results = list(ex.map(square, range(5)))

25. Example — log monitor script

# Monitor error lines in a log file, live
import time
logfile = "/var/log/app.log"
with open(logfile) as f:
    f.seek(0, 2)  # jump to end of file
    while True:
        line = f.readline()
        if not line:
            time.sleep(1)
            continue
        if "ERROR" in line:
            print(line.strip())

  • Handle exceptions gracefully

  • Test scripts before production
  • Follow PEP 8 style

29. Interview quick Q&A

Q: Difference between list and tuple?  A: List is mutable; tuple is immutable.
Q: What are *args and **kwargs?  A: *args = variable positional arguments; **kwargs = variable keyword arguments.
Q: What is subprocess used for?  A: To run shell commands and automate system tasks from Python.
Q: How do you read environment variables?  A: os.getenv("VAR_NAME")
Q: Difference between threading and multiprocessing?  A: Threads share memory (good for I/O-bound); processes have separate memory (good for CPU-bound, side-step the GIL).
